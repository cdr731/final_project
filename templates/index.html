Fruit Image Classification

Project Summary

  • Kaggle Dataset:
    • 74K images of 109 fruits & vegetables
    • Each image is a 30,000 dimensional vector (100 pixels x 100 pixels x 3 colors)
  • Image Feature Extraction Using Transfer Learning
  • Unsupervised Learning
    • Similar image retrieval using K-nearest neighbors
    • The KD-Tree data structure was used to speed-up K-NN queries
  • Supervised Learning
    • Train/Test Set: 58,266/19,548 images
    • Logistic Regression
    • Neural Network
