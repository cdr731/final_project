Fruit Image Classification

Project Summary

The purpose of this project is to create a machine learning algorithm which can determine the type of fruit an image presents. The database has 74681 images of 109 fruits/vegetables. We ran supervised and unsupervised machine learning algorithms to determine the fruit contained in each 100x100 image. We used Logistic Regression, and our testing score was 0.989
